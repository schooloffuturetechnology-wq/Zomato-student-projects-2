# Zomato Student Projects 2

A Zomato-inspired landing page built with HTML and CSS.

## Project Files

- `index.html` - main deployment entry file
- `zomato.html` - original project page file
- `zom.mp4` - hero background video
- `zom.png` - project preview image

## Preview

![Zomato Clone Preview](zom.png)

## Features

- full-screen video hero section
- app store download badges
- branded cards section
- hover scale effects
- responsive layout

## Run Locally

Open `index.html` in any web browser.
